## Warm-up - one choice or short loops

- **[Task 1: Running total at the bake sale]({{ site.baseurl }}/tasks/2-1/)** - Add sale amounts until the cashier enters 0; print the total takings.
- **[Task 2: Highest temperature this week]({{ site.baseurl }}/tasks/2-2/)** - Read five daily high temperatures; print the largest.
- **[Task 3: Staff room keypad]({{ site.baseurl }}/tasks/2-3/)** - Keep asking for a password until the user enters `python123`.
- **[Task 4: Counting even raffle tickets]({{ site.baseurl }}/tasks/2-4/)** - Count how many even ticket numbers were entered; stop when the user enters -1.
- **[Task 5: Zoo ticket price]({{ site.baseurl }}/tasks/2-5/)** - One visitor’s age → print child, adult, or senior price using `if` / `elif` / `else`.
- **[Task 6: Report card letter]({{ site.baseurl }}/tasks/2-6/)** - Turn one percentage mark into a letter grade (A–F) with `elif` ladder.
- **[Task 7: Freezing day on the farm]({{ site.baseurl }}/tasks/2-7/)** - Read one Celsius temperature; say if water would freeze outside.
- **[Task 8: Odd or even order total]({{ site.baseurl }}/tasks/2-8/)** - One integer: say if it is odd or even (useful for splitting orders).
- **[Task 9: Bus departure countdown]({{ site.baseurl }}/tasks/2-9/)** - Print a countdown from N down to 1 using a `for` loop.
- **[Task 10: Sticker sheet total]({{ site.baseurl }}/tasks/2-10/)** - Sum the integers 1 + 2 + … + N for a user-chosen N using a loop.

## Loops with decisions - money, games, and data

- **[Task 11: Strawberry stand checkout]({{ site.baseurl }}/tasks/2-11/)** - Until the customer pays enough: show change or exact payment; use 1 decimal for change.
- **[Task 12: Valid marks only]({{ site.baseurl }}/tasks/2-12/)** - Average exam marks between 50 and 100; stop at -5; ignore invalid marks.
- **[Task 13: Waiting for a lucky 7]({{ site.baseurl }}/tasks/2-13/)** - Random 1–10 until 7; print each roll and count attempts.
- **[Task 14: Coin race to ten]({{ site.baseurl }}/tasks/2-14/)** - Flip until heads or tails reaches 10; print winner and total flips.
- **[Task 15: Higher–lower guessing game]({{ site.baseurl }}/tasks/2-15/)** - Classic 1–100 secret number until correct (like the Unit 2 project).
- **[Task 16: ATM PIN (three tries)]({{ site.baseurl }}/tasks/2-16/)** - Lock out after three wrong PIN attempts; success prints welcome message.
- **[Task 17: Five-question mental maths quiz]({{ site.baseurl }}/tasks/2-17/)** - Ask five addition questions; score how many were correct.
- **[Task 18: Charity jar total]({{ site.baseurl }}/tasks/2-18/)** - Add donations until user enters 0; print total raised.
- **[Task 19: Shop times table poster]({{ site.baseurl }}/tasks/2-19/)** - Print the multiplication table for one integer from 1×N to 12×N.
- **[Task 20: Factorial orderings]({{ site.baseurl }}/tasks/2-20/)** - Compute N! for a small positive N using a loop (no `math.factorial`).
- **[Task 21: Weekly step average]({{ site.baseurl }}/tasks/2-21/)** - Seven inputs; print average daily steps (integer division OK or float).
- **[Task 22: Savings goal]({{ site.baseurl }}/tasks/2-22/)** - Each week add savings until balance reaches or passes a goal amount.
- **[Task 23: Strong enough Wi‑Fi password]({{ site.baseurl }}/tasks/2-23/)** - Reject passwords shorter than 8 characters until a valid one is entered.
- **[Task 24: Guess with seven tries]({{ site.baseurl }}/tasks/2-24/)** - Number 1–50; player has at most 7 guesses; win or lose message.
- **[Task 25: Rock, paper, scissors - first to three]({{ site.baseurl }}/tasks/2-25/)** - Best-of-style loop until one player reaches 3 wins (user vs computer).

## Applied scenarios - shops, travel, and patterns

- **[Task 26: BMI and health band]({{ site.baseurl }}/tasks/2-26/)** - Compute BMI from height (m) and weight (kg); print category; re-prompt if inputs invalid.
- **[Task 27: US road trip temperatures]({{ site.baseurl }}/tasks/2-27/)** - Convert seven Celsius forecasts to Fahrenheit each day.
- **[Task 28: Stock on the shelf]({{ site.baseurl }}/tasks/2-28/)** - Start with stock count; subtract each sale until stock hits 0 or user quits.
- **[Task 29: Bus passenger count]({{ site.baseurl }}/tasks/2-29/)** - Track riders: positive = board, negative = leave; stop at sentinel; never below 0.
- **[Task 30: FizzBuzz inventory labels]({{ site.baseurl }}/tasks/2-30/)** - Print 1..N replacing multiples of 3 and 5 with words (classic FizzBuzz).
- **[Task 31: Is this number prime?]({{ site.baseurl }}/tasks/2-31/)** - Test one integer ≥ 2: print prime or not using a trial division loop.
- **[Task 32: Collatz hailstone length]({{ site.baseurl }}/tasks/2-32/)** - Start from n: if even n//2 else 3n+1; count steps until you reach 1.
- **[Task 33: Dice doubles at the café]({{ site.baseurl }}/tasks/2-33/)** - Roll two dice until they match; count how many rolls.
- **[Task 34: Simple pay-down balance]({{ site.baseurl }}/tasks/2-34/)** - Subtract fixed payment each month until balance ≤ 0; print month count.
- **[Task 35: Coffee stamp card]({{ site.baseurl }}/tasks/2-35/)** - 10 stamps earn a free drink; simulate until two free rewards are earned.
- **[Task 36: Elevator floor hops]({{ site.baseurl }}/tasks/2-36/)** - Start at floor 1; repeatedly go to requested floor until user enters 0.
- **[Task 37: Guess the product]({{ site.baseurl }}/tasks/2-37/)** - Secret A and B 1–10; user guesses product until correct.
- **[Task 38: Live class average]({{ site.baseurl }}/tasks/2-38/)** - After each valid mark (0–100), print running average; -1 to stop.
- **[Task 39: Café lunch order]({{ site.baseurl }}/tasks/2-39/)** - Simple menu: loop choose 1–3 or 0 to pay; accumulate bill.
- **[Task 40: Fibonacci rabbit pairs (small n)]({{ site.baseurl }}/tasks/2-40/)** - Print first n Fibonacci numbers (1, 1, 2, 3, 5, …) with a loop.

## Stretch - nested loops, math, and trickier control flow

- **[Task 41: Greatest common divisor]({{ site.baseurl }}/tasks/2-41/)** - Euclid’s algorithm with a while loop for two positive integers.
- **[Task 42: Powers by repeated multiplication]({{ site.baseurl }}/tasks/2-42/)** - Compute base^exponent with repeated multiplication (exponent ≥ 0); do not use `**` or `pow`.
- **[Task 43: Count digits in a sale ID]({{ site.baseurl }}/tasks/2-43/)** - Positive integer: how many digits (repeated // 10).
- **[Task 44: Reverse digits of a sale code]({{ site.baseurl }}/tasks/2-44/)** - Build reversed integer with % 10 and // 10 in a loop.
- **[Task 45: Sum of squares pyramid]({{ site.baseurl }}/tasks/2-45/)** - Compute 1² + 2² + … + n² for user n.
- **[Task 46: Guess my number with play again]({{ site.baseurl }}/tasks/2-46/)** - Wrap the guessing game: after win, ask y/n to play again (outer while).
- **[Task 47: Battery drain simulation]({{ site.baseurl }}/tasks/2-47/)** - Start 100%; subtract random drain until ≤ 0; count hours.
- **[Task 48: Star triangle banner]({{ site.baseurl }}/tasks/2-48/)** - Print a triangle of `*` with n rows (row i has i stars).
- **[Task 49: Full multiplication square]({{ site.baseurl }}/tasks/2-49/)** - Double loop: for rows 1–12 and cols 1–12 print row×col in a grid (tab-separated OK).
- **[Task 50: Delivery van route checker]({{ site.baseurl }}/tasks/2-50/)** - Sum package weights until over capacity or user ends; report status.
