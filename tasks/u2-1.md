# 2.2 Loops

**Loops** repeat a block of code. That way we can keep asking for a guess until the user gets it right, or run a game until it’s over.

## while loop

A `while` loop runs **while** a condition is True:

```python
count = 0
while count < 3:
    print("Hello")
    count = count + 1
# Prints Hello three times
```
