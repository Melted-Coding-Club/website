## Lists - building, looping, and slicing

- **[Task 1: Weekly lunch menu]({{ site.baseurl }}/tasks/3-1/)** - Print each weekday with the meal from a fixed list (indexes 0-4).
- **[Task 2: Lap times summary]({{ site.baseurl }}/tasks/3-2/)** - From a list of lap times (seconds), print the fastest lap and the average.
- **[Task 3: Build a playlist]({{ site.baseurl }}/tasks/3-3/)** - Append song titles until the user enters a blank line; print the playlist.
- **[Task 4: Marks in a list]({{ site.baseurl }}/tasks/3-4/)** - Read marks into a list until -1; print count and average.
- **[Task 5: Unique sign-up list]({{ site.baseurl }}/tasks/3-5/)** - Append a name only if it is not already in the list.
- **[Task 6: Team roster]({{ site.baseurl }}/tasks/3-6/)** - Add player names until 'done'; print how many players.
- **[Task 7: Cheapest item index]({{ site.baseurl }}/tasks/3-7/)** - After entering five prices, print the position (1-5) of the lowest price.
- **[Task 8: Double every level score]({{ site.baseurl }}/tasks/3-8/)** - Build a new list where each value is twice the original (given list in code).
- **[Task 9: Merge two stock shelves]({{ site.baseurl }}/tasks/3-9/)** - Add two same-length lists element-wise into a third list.
- **[Task 10: Reading list slices]({{ site.baseurl }}/tasks/3-10/)** - Print first three, last two, and a middle slice from a fixed list.

## Dictionaries - lookup, counting, and updates

- **[Task 11: Cafe menu order total]({{ site.baseurl }}/tasks/3-11/)** - Use a dict of item to price; loop orders until done; print total.
- **[Task 12: Capital city lookup]({{ site.baseurl }}/tasks/3-12/)** - Dictionary country to capital; loop until quit.
- **[Task 13: Word tally from a sentence]({{ site.baseurl }}/tasks/3-13/)** - Split a line into words; count each word in a dictionary.
- **[Task 14: Letter frequency]({{ site.baseurl }}/tasks/3-14/)** - Count how often each character appears in a word.
- **[Task 15: Inventory commands]({{ site.baseurl }}/tasks/3-15/)** - Dict of product to count; process add, show, and stop commands.
- **[Task 16: Attendance tally]({{ site.baseurl }}/tasks/3-16/)** - Start with an empty dict; each entered name increases their day count.
- **[Task 17: Simple phrasebook]({{ site.baseurl }}/tasks/3-17/)** - Translate words using a dict; unknown words get a default message.

## Mixed - lists and dictionaries together

- **[Task 18: Election-style tally]({{ site.baseurl }}/tasks/3-18/)** - Collect votes in a list then count with a dict to find the winner.
