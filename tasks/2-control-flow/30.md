# Unit 2 Task 30 - FizzBuzz inventory labels

[← Unit 2 tasks]({{ site.baseurl }}/tasks/2-control-flow/) · [Solution]({{ site.baseurl }}/tasks/2-control-flow/30-solution/)

---


Warehouse labels use codes **1** to **N**. For each integer:

- If divisible by **3** and **5**, print **`FizzBuzz`**
- Else if by **3**, print **`Fizz`**
- Else if by **5**, print **`Buzz`**
- Else print the number

Ask for **N** (positive). Use a `for` loop from 1 to N.

**Skills:** `%`, `if` / `elif`, `for`.
