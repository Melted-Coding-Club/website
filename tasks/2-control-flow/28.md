# Unit 2 Task 28 - Stock on the shelf

[← Unit 2 tasks]({{ site.baseurl }}/tasks/2-control-flow/) · [Solution]({{ site.baseurl }}/tasks/2-control-flow/28-solution/)

---


A shop starts with **S** items. Repeatedly ask how many were **sold** this hour (integer). Subtract from stock.

- If a sale would make stock **negative**, print **`Not enough stock`** and **do not** change stock.
- If the user enters **0** as sold, treat that as **closing time** and stop.
- When stock reaches **0**, print **`Shelf empty`** and stop.

Print remaining stock whenever you like (e.g. after each valid sale).

**Skills:** `while`, guards on subtraction.
