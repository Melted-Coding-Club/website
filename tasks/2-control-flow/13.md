# Unit 2 Task 13 - Waiting for a lucky 7

[← Unit 2 tasks]({{ site.baseurl }}/tasks/2-control-flow/) · [Solution]({{ site.baseurl }}/tasks/2-control-flow/13-solution/)

---


Use `import random`. Repeatedly pick `random.randint(1, 10)`:

- Print each value.
- Stop when **7** appears.
- Then print how many **attempts** it took (including the 7).

**Skills:** `while True`, `break`, `random.randint`, counter.
