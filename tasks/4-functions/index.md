## Warm-up - `def`, parameters, and `print`

- **[Task 1: Club welcome message]({{ site.baseurl }}/tasks/4-functions/1/)** - `welcome(name)` prints a one-line greeting using the parameter.
- **[Task 2: Screen divider]({{ site.baseurl }}/tasks/4-functions/2/)** - `divider()` prints a fixed line of characters (no parameters).

## Returning numbers

- **[Task 3: Price after discount]({{ site.baseurl }}/tasks/4-functions/3/)** - `discounted_price(price, percent_off)` returns the sale price.
- **[Task 4: Zoo ticket total]({{ site.baseurl }}/tasks/4-functions/4/)** - `ticket_total(adults, children)` returns total cost from counts.
- **[Task 5: Film length in minutes]({{ site.baseurl }}/tasks/4-functions/5/)** - `total_minutes(hours, minutes)` returns one integer.

## Booleans and decisions

- **[Task 6: Tall enough to ride]({{ site.baseurl }}/tasks/4-functions/6/)** - `can_ride(height_cm)` returns whether height is at least 120 cm.
- **[Task 7: Is it a vowel?]({{ site.baseurl }}/tasks/4-functions/7/)** - `is_vowel(ch)` returns `True` for a single vowel letter (ignore case).
- **[Task 8: Pass mark]({{ site.baseurl }}/tasks/4-functions/8/)** - `passed(mark)` returns `True` if the mark is 40 or more.

## Loops inside functions

- **[Task 9: Small factorial]({{ site.baseurl }}/tasks/4-functions/9/)** - `factorial(n)` returns n! for n between 1 and 10 (use a loop, not `math.factorial`).
- **[Task 10: Sum from 1 to n]({{ site.baseurl }}/tasks/4-functions/10/)** - `sum_to(n)` returns 1 + 2 + … + n for n ≥ 1.
- **[Task 11: Count divisors]({{ site.baseurl }}/tasks/4-functions/11/)** - `count_divisors(n)` returns how many integers from 1 to n divide n evenly.

## Putting it together

- **[Task 12: Three addition questions]({{ site.baseurl }}/tasks/4-functions/12/)** - `ask_addition()` asks one question and returns `True` if correct; `main()` runs three rounds and prints the score.
