# Unit 3 Task 15 - Inventory commands

[← Unit 3 tasks]({{ site.baseurl }}/tasks/3/) - [Solution]({{ site.baseurl }}/tasks/3-15-solution/)

---



Start with a dictionary of **product name -> quantity** (integers), e.g. `{"pencil": 20, "eraser": 15}`.

Loop reading **commands**:

- `show` - print all products and quantities.
- `add PRODUCT QTY` - split the line; increase that product's count by QTY (assume product exists and QTY is a valid int).
- `stop` - exit the loop.

You may assume commands are well-formed for `add`.

**Skills:** dict mutation, `split()`, string commands.
