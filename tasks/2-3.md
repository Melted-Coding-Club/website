# Unit 2 Task 3 - Staff room keypad

[← Unit 2 tasks]({{ site.baseurl }}/tasks/2/) · [Solution]({{ site.baseurl }}/tasks/2-3-solution/)

---


The staff room door only opens when someone types the correct password: **`python123`**.

Write a program that:

- Repeatedly asks: `Enter password: `
- Keeps asking until the exact password is entered.
- Then prints: `Access granted`

Use a `while` loop and string comparison (`==`).

**Skills:** `while`, strings, `input()`.
