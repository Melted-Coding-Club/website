# Unit 2 Task 39 - Café lunch order

A café menu:

1. Sandwich - **$6**  
2. Soup - **$4**  
3. Drink - **$2**  
0. **Done** ordering  

Show the menu in a loop. Add prices to a **total** until the user chooses **0**. Then print **`Your total is $X`** (you can use `round(..., 2)`).

**Skills:** `while`, menu `if` / `elif`, accumulator.
