# AI use on this site

This website is a **personal teaching resource** I put together for my students. I am **not trying to hide** that parts of it were built or refined with help from **AI tools**, mainly **[Cursor](https://cursor.com/)** (an editor that includes coding assistants).

Using AI here is a practical choice: it speeds up drafting lessons, tasks, tutorials, and site structure so I can focus on **what** we teach and **how** it fits my class-not on pretending everything was typed by hand from scratch.

**What that does and does not mean**

- **Does mean:** Some wording, examples, exercise ideas, and technical edits may have started from or been improved through AI-assisted suggestions. I still **curate, check, and align** content with what I want students to learn.
- **Does not mean:** Students should copy AI output without thinking. In class we still stress **understanding**, **testing**, and **your own reasoning**-the same standards I apply when using these tools to prepare materials.

If you have questions about how we use AI in coursework versus for this site, ask in class. Transparency matters.
